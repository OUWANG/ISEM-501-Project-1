ISEM 501 SYLLABUS

We structure what we learn

Hardware, Software, HTML and CSS

Basic software concepts are explored within this context by developing simple web sites using HTML and then using JavaScript, Java applets and XML to introduce more sophisticated features.

Databases, SQL, and XML

The student has an opportunity to learn database technologies and run simple database queries using SQL.

Telecommunication, xNet, and JavaScript

The emphasis is on the Internet, broadband wired and wireless networks, classical Web, Semantic Web, XML, Web 2.0, social networking, and mobile computing.

Quizzes, Projects, and Exams

This course provides the basic background in the rapidly advancing field of information and communication technologies (ICTs).
